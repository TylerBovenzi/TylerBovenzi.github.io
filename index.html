About Me

0x00A0

Engineer by training,
flight test by trade.

I'm an avionics engineer at Stratolaunch, where I lead data systems and telemetry operations for the Talon-A autonomous hypersonic vehicle program. On any given mission that means configuring instrumentation, running the telemetry console in Mission Control, managing antenna tracking, and rerouting data streams through range assets in real time, then leading the ground team out to recover the vehicle after it lands.

Away from the flight line I write automation tools and build embedded systems, things like a DAS auto-configuration tool, a fully autonomous remote weather station, and an FPGA-based high-bitrate PCM telemetry playback system. I also led the satellite communication installation on the carrier aircraft, start to finish.

Before Stratolaunch I studied Computer Engineering at Cal Poly SLO, where I built GPS trackers, solar UAVs, and robotics control boards. The thread running through all of it: I like making things that actually have to work.
